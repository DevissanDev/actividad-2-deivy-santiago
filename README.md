# Actividad 2 - Deivy Santiago

Este proyecto es un sitio web de dos paginas sobre JavaScript, desarrollado con HTML, CSS y Bootstrap.

## Contenido del sitio

- Primera pagina: explica que es JavaScript, para que sirve y por que aprenderlo.
- Segunda pagina: presenta la historia de JavaScript, su origen, evolucion y actualidad.

## Estructura del proyecto

- index.html
- pagina.html
- css/iniciostyle.css
- css/paginastyle.css
- imagenes/

## Caracteristicas implementadas

- Diseno responsive con Bootstrap.
- Navegacion entre secciones con anclas.
- Video incrustado de YouTube.
- Uso de imagenes locales para reforzar el contenido.
- Footer con enlaces utiles y datos de contacto.
