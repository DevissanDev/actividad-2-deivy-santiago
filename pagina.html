JavaScript nacio en 1995, en una epoca en la que las paginas web empezaban a volverse mas complejas. Los formularios y la interaccion crecian, y HTML por si solo ya no alcanzaba para ofrecer una buena experiencia.

1995
Inicio de JavaScript

Su gran ventaja fue ejecutarse directamente en el navegador, sin compilaciones complicadas. Eso permitio validar datos y responder a acciones del usuario de forma inmediata.

Creador y origen

Brendan Eich
Netscape, 1995

Brendan Eich creo el lenguaje en Netscape para resolver las limitaciones del desarrollo web de ese momento. El nombre inicial fue LiveScript, y despues paso a llamarse JavaScript para aprovechar la popularidad de Java.

Aunque comparten ciertas similitudes de sintaxis, JavaScript y Java son tecnologias diferentes.

Estandares y evolucion

1997
ECMAScript

En 1996 Internet Explorer incorporo soporte para JavaScript y acelero su adopcion. Luego, en 1997, Ecma International estandarizo el lenguaje bajo el nombre tecnico ECMAScript.

Con los anos llegaron frameworks como Angular y React, y Node.js lo llevo tambien al backend.

JavaScript en la actualidad

Hoy JavaScript es parte esencial de la web moderna junto con HTML y CSS. Se usa en navegadores, servidores, apps moviles y muchos otros entornos de desarrollo.

Tambien es uno de los lenguajes mas demandados laboralmente. Su facilidad para empezar, su enorme comunidad y su versatilidad lo convierten en una excelente opcion para aprender programacion.
